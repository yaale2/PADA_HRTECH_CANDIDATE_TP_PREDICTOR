# PADA_HRTECH_CANDIDATE_TP_PREDICTOR
# Прогнозирование увольнения сотрудника на основе анализа структурированных данных резюме

Гибридная нейросетевая система (LSTM + Attention) для прогнозирования риска увольнения сотрудников на основе анализа иерархических данных резюме.

## Основные характеристики

- **C-index:** 0.76
- **Архитектура:** LSTM + Attention + DeepSurv Head
- **Векторизация:** Sentence-BERT (paraphrase-MiniLM-L3-v2)
- **Обработка цензурированных данных:** Cox Partial Likelihood
